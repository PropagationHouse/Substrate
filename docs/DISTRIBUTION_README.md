# Tiny Pirate Installation Guide 
 
## Installation 
 
1. Run the installer and follow the on-screen instructions. 
2. After installation, you may need to install Ollama separately if you want to use local LLM functionality. 
   - Download Ollama from https://ollama.com/ 
   - Follow the installation instructions on the Ollama website. 
 
## First Run 
 
When you first run Tiny Pirate: 
 
1. The application will automatically install required Python dependencies. 
2. You'll be prompted to configure your settings. 
3. For full functionality, ensure Ollama is running in the background. 
 
## Features 
 
- **Chat Interface**: Interact with local LLMs through Ollama 
- **Voice Synthesis**: Text-to-speech capabilities 
- **Profile Management**: Multiple user profiles with customizable settings 
- **Screenshot Analysis**: Capture and analyze screen content 
- **Note Creation**: Automated and manual note-taking capabilities 
- **Aurora Forecast**: Check aurora borealis forecasts and predictions (Note: General weather functionality has been removed) 
 
## Troubleshooting 
 
If you encounter issues: 
 
1. Ensure Python 3.8+ is installed and in your PATH 
2. Check that Ollama is running if you're using local LLM features 
3. Restart the application after installing any missing dependencies
