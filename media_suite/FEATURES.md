# Media Planning Suite - Feature Overview

## 🎨 Glassmorphic Design System

The application features a premium glassmorphic UI with:
- Frosted glass effects with backdrop blur
- Gradient backgrounds and animated floating shapes
- Smooth transitions and hover effects
- Responsive design for all screen sizes
- Dark theme optimized for long work sessions

## 📋 Core Features

### 1. Business Profile Management
**Create and manage multiple business profiles for agencies or individual businesses**

- **Business Information**
  - Company name and industry selection
  - Detailed business description
  - Website URL
  - Target audience definition
  - Products/services offerings

- **Brand Identity**
  - Brand color palette management
  - Font selection and management
  - Logo upload and storage
  - Consistent brand application across all outputs

### 2. Media Planning System
**Comprehensive campaign planning from start to finish**

- **Plan Creation**
  - Campaign title and description
  - Start and end date scheduling
  - Budget allocation and tracking
  - Status management (draft, active, completed)
  - Business profile association

- **Plan Organization**
  - Multiple plans per business
  - Visual plan cards with status indicators
  - Quick access to plan details
  - Timeline visualization

### 3. Shot List Management
**Professional production planning for photographers and videographers**

- **Shot Details**
  - Shot title and type (wide, medium, close-up, detail, action, interview, b-roll, product)
  - Location specification
  - Time requirements
  - Equipment lists
  - Detailed descriptions and notes
  - Reference image uploads

- **Production Features**
  - Completion tracking
  - Shot numbering and organization
  - Equipment checklists
  - Location planning
  - Export to PDF for crew

### 4. Storyboard Creation
**Visual planning for video content**

- **Scene Management**
  - Scene numbering and sequencing
  - Scene titles and descriptions
  - Visual reference uploads
  - Scene reordering capabilities

- **Production Notes**
  - Visual notes (camera angles, composition, lighting)
  - Audio notes (dialogue, music, sound effects)
  - Duration specifications
  - Scene-by-scene breakdown

- **Preview Mode**
  - Full-screen storyboard preview
  - Navigate between scenes
  - Professional presentation format

### 5. Media Calendar
**Schedule and track content releases**

- **Calendar Features**
  - Monthly calendar view
  - Content scheduling
  - Channel assignment (social media, website, email, etc.)
  - Content type categorization
  - Status tracking

- **Content Planning**
  - Multiple content items per day
  - Visual calendar grid
  - Quick event creation
  - Asset attachment

### 6. Automated Research System
**Industry insights and competitor analysis**

- **Web Research**
  - Automated web scraping
  - Topic-based research queries
  - Top 10 results extraction
  - Source tracking

- **Research Management**
  - Save research findings
  - Associate with business profiles
  - Search history
  - Content snippets and links

### 7. Brand Asset Management
**Centralized asset storage and organization**

- **Asset Types**
  - Logos and brand marks
  - Color palettes
  - Font files
  - Templates
  - Brand images
  - Brand guidelines documents

- **Asset Features**
  - File upload and storage
  - Preview capabilities
  - Download functionality
  - Metadata management
  - Business profile association

### 8. Export & Collaboration
**Production-ready documents for teams**

- **Export Formats**
  - Comprehensive package (all plan details)
  - Shot list only
  - Storyboard only
  - Media calendar
  - Creative brief

- **Export Features**
  - Print-optimized layouts
  - PDF generation
  - Professional formatting
  - Brand header inclusion
  - Team instructions

## 🛠️ Technical Highlights

- **Backend**: Node.js + Express
- **Frontend**: Vanilla JavaScript (no framework overhead)
- **Styling**: TailwindCSS with custom glassmorphic components
- **Database**: SQLite with full-text search
- **File Processing**: Sharp for image optimization
- **Web Scraping**: Puppeteer for automated research
- **Security**: Input validation, file type restrictions, SQL injection prevention
